* =bin/shared/mailer/=
A simple mailer to send out batch emails from YAML configuration files using
SMTP.

** Usage
#+BEGIN_SRC ruby
mailer = Mailer.new(path_to_yaml_file)
mailer.send!
#+END_SRC

** YAML File Format
#+BEGIN_SRC yaml
emails:
  -
    subject: 'Email Subject 1'
    to: 'email1@gmail.com'
    sender: 'sender.email@gmail.com'
    template: 'sample'
    variables:
      foo: 'bar'

  -
    subject: 'Email Subject 2'
    to: 'email2@gmail.com'
    sender: 'sender.email@gmail.com'
    template: 'sample'
    variables:
      foo: 'bar2'
#+END_SRC
